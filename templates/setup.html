API Token Required

Configure your Readwise Reader API token to get started

Setup Instructions:
  1. Get your API token from Readwise Access Token
  2. Choose one of the methods below to configure your token

Windows (PowerShell):

$env:READWISE_TOKEN = "your_api_token_here"

Windows (Command Prompt):

set READWISE_TOKEN=your_api_token_here

Linux/macOS:

export READWISE_TOKEN=your_api_token_here
After setting the environment variable, restart the web server for changes to take effect.

Create a file named .readwise_token in the project root directory:

Windows:

echo your_api_token_here > .readwise_token

Linux/macOS:

echo "your_api_token_here" > .readwise_token
After creating the file, refresh this page to continue.

Use the command-line tool to save your token:

python cli.py setup-token --token your_api_token_here
This will create the .readwise_token file automatically.
Your API token is stored locally and never shared with third parties.
